Strings

1) Concatenar con la coma

Al poner ',' se concatena las cadenas con variables
Existe un expaciado extra en la impresión.

Usando:

const nombre = "Chris";
console.log('Hola ',nombre,'. Su nombre tiene: ',nombre.length,' letras.');

2) Concatenar con el signo +

Al poner '+' se concatena las cadenas con variables
No existe un expaciado extra en la impresión.

Usando:

const nombre = "Chris";
console.log('Hola '+nombre+'. Su nombre tiene: '+nombre.length+' letras.');

3) Concatenar con el signo `

Al poner '`' se concatena las cadenas con variables Es lo recomendado a usar

Usando:

const nombre = "Chris"
console.log(`Hola ${nombre}. Su nombre tiene: ${nombre.length} letras.`);

4) Convertir un texto a minúscula

Usando:

const texto1 = 'MARIA';
console.log(texto1.toLowerCase());

5) Convertir un texto a mayúscula

Usando:

const texto2 = 'carlos';
console.log(texto2.toUpperCase());

6) Verifica que exista el string en la cadena

Usando:

const texto3 = 'lorem tres cuatro cinco';
console.log(texto3.includes('cinco'));

7) Elimina los espacios de inicio y fin con trim

Usando:

const texto4 = ' lorem tres cuatro cinco ';
console.log(texto4);
console.log(texto4.trim());

8) Separa las cadenas a partir un espacio y lo visualiza en un array con Split

Usando:

const texto5 = 'lorem tres cuatro cinco';
console.log(texto5.split(' '));

9) Separa las cadenas a partir un espacio y lo visualiza en un array con Split

Usando:

const texto6 = "Dani Fhn. Filth. Test. Juan";
const splitTexto6 = texto6.split('.')[2];
console.log(splitTexto6);

10) Separa las cadenas a partir un espacio y lo visualiza en un array con Split

Usando:

const texto7 = "Dani.Chris";
const splitTexto7 = texto7.split('.');
console.log(splitTexto7);

11) Imprime la primera letra de la cadena con Substring

Usando:

const texto8 = "dani Fhn Filth asd, asddsf";
const splitTexto8 = texto8.substring(0, 1);
console.log(splitTexto8);

12) Imprime la primera letra de la cadena y lo convierte a mayúscula

Usando:

const texto9 = "dani Fhn Filth asd, asddsf";
const splitTexto9 = texto9.substring(0, 1).toUpperCase();
console.log(splitTexto9);

13) Imprime la primera letra de la cadena y lo convierte a mayúscula

Usando:
