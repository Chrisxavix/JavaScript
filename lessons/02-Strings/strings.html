Strings

1.- Concatenar con la coma 2.- Concatenar con el signo + 3.- Concatenar con el signo ` 4.- toLowerCase(): convertir un texto a minúscula 5.- toUpperCase(): convertir un texto a mayúscula 6.- includes(): verifica que exista el string en la cadena 7.- trim(): elimina los espacios de inicio y fin 8.- split(): separa las cadenas a partir un valor 9.- substring(1): elimina la primera letra de la cadena 10.- replace(): reemplaza un string por otro

1) Concatenar con la coma

Al poner ',' se concatena las cadenas con variables
Existe un expaciado extra en la impresión.

Usando:

const nombre = "Chris";
console.log('Hola ',nombre,'. Su nombre tiene: ',nombre.length,' letras.');

2) Concatenar con el signo +

Al poner '+' se concatena las cadenas con variables
No existe un expaciado extra en la impresión.

Usando:

const nombre = "Chris";
console.log('Hola '+nombre+'. Su nombre tiene: '+nombre.length+' letras.');

3) Concatenar con el signo `

Al poner '`' se concatena las cadenas con variables Es lo recomendado a usar

Usando:

const nombre = "Chris"
console.log(`Hola ${nombre}. Su nombre tiene: ${nombre.length} letras.`);

4) toLowerCase(): convertir un texto a minúscula

Usando:

const texto1 = 'MARIA';
console.log(texto1.toLowerCase());

5) toUpperCase(): convertir un texto a mayúscula

Usando:

const texto2 = 'carlos';
console.log(texto2.toUpperCase());

6) includes(): verifica que exista el string en la cadena

Si encuentra responde con true, caso contrario con false

Usando:

const texto3 = 'lorem tres cuatro cinco';
console.log(texto3.includes('cinco'));

7) trim(): elimina los espacios de inicio y fin

Usando:

const texto4 = ' lorem tres cuatro cinco ';
console.log(texto4);
console.log(texto4.trim());

8) split(): separa las cadenas a partir un valor

Usando:

const texto5 = 'lorem tres cuatro cinco';
console.log(texto5.split(' '));

8.1) split('.')[1]: imprime el primer string despues del "." a partir de la posición

Usando:

const texto6 = "Dani Fhn. Filth. Test. Juan";
const splitTexto6 = texto6.split('.')[1];
console.log(splitTexto6);

9) substring(1): elimina la primera letra de la cadena

Usando:

const texto10 = "Dani Fhn Filth asd, asddsf";
const splitTexto10 = texto10.substring(1);
console.log(splitTexto10);

9.1) substring(0, 1): imprime la primera letra de la cadena con substring

Usando:

const texto8 = "dani Fhn Filth asd, asddsf";
const splitTexto8 = texto8.substring(0, 1);
console.log(splitTexto8);

9.2) substring(0, 1).toUpperCase(): imprime la primera letra de la cadena y lo convierte a mayúscula

Usando:

const texto9 = "dani Fhn Filth asd, asddsf";
const splitTexto9 = texto9.substring(0, 1).toUpperCase();
console.log(splitTexto9);

10) replace(): reemplaza un string por otro

Toma una parte del texto y lo reemplaza, dando una cadena de texto nueva

Usando:

const arrayReplace = 'www.texto.com?id=3';
const resultReplace = arrayReplace.replace('texto.', 'maggot-code.')
console.log(arrayReplace, 'original');
console.log(resultReplace, 'modificado');
