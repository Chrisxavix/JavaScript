Operators

1.- Comparación de datos

Dos iguales (==) se comparan los valores
Tres iguales (===) se comparan los tipos de datos y el valor

Usando:


console.log('7'==7);
console.log('7'===7);
console.log(7===7);

2.- Operador (+=), incrementa

Es como tener (i = i + 2;) igual a (i += 2;)

Usando:


let valor1 = 1;
valor1 += 2;
console.log(valor1);

3.- Realizar cualquier operación (*=)

Usando:


let valor2 = 8;
valor2 *= 2;
console.log(valor2);

4.- Incremento binario, como un contador en +1

Usando:


let valor3 = 3;
valor3 ++;
console.log(valor3);

5.- Incremento binario, lo mismo que el anterior

Usando:


let valor4 = 2;
++valor4;
console.log(valor4);

6.- Operador (!), que denota negación

Usando:


let valor5 = true;
console.log(!valor5);
